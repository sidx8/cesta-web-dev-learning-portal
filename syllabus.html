There are three types of web developers it -
1. Front end developers — one who develops the website and the visuals using HTML, CSS and JavaScript.
2. Back end developers — one who develops the actions on a website or web apps using programming languages like JavaScript, Python, Php etc.
3. Full stack developer — one who can develop all layers of the software, has the skill set of both front-end and back-end developers.
